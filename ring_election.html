*To start an election, click on a node.
*To crash a node, right-click on it. To reactivate, right-click on it again.

How to Use the Simulator:

  1. Enter the structure of the Ring. This is done by entering
    between 3 and 8 distinct and non-negative numbers.
  2. Next, select a speed for the simulation to run at. If 'Step' is
    selected, press the 'Step' button to continue the simulation.
  3. To reset the simulator, press the 'Reset' button next to the
    'Start' button.
